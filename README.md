## Overview

The **Project Dashboard** is a comprehensive Excel-based project management solution built with VBA. It provides real-time tracking, visualization, and analysis of project portfolios with an executive-level dashboard interface. Perfect for project managers, team leads, and executives who need instant insights into project performance.

![Dashboard Preview](https://github.com/bilgambuyu/Project-Tracker/blob/main/screenshot.gif)

## Key Features

- **Real-Time Dashboard** - Live updating dashboard with key performance indicators
- **Interactive Charts** - Donut charts, bar charts, and performance metrics
- **Regional Analysis** - Track projects across Americas, Europe, Asia, Middle East, Africa
- **Manager Performance** - Individual manager metrics with color-coded performance
- **Real-Time Updates** - Instant dashboard refresh when data changes
- **Data Validation** - Dropdown lists for status, region, and managers
