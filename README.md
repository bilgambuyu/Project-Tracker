## 🎯 Overview

The **Executive Project Dashboard** is a comprehensive Excel-based project management solution built with VBA. It provides real-time tracking, visualization, and analysis of project portfolios with an executive-level dashboard interface. Perfect for project managers, team leads, and executives who need instant insights into project performance.

![Dashboard Preview](https://via.placeholder.com/800x400?text=Executive+Dashboard+Preview)

## ✨ Key Features

- **📊 Real-Time Dashboard** - Live updating dashboard with key performance indicators
- **📈 Interactive Charts** - Donut charts, bar charts, and performance metrics
- **🌍 Regional Analysis** - Track projects across Americas, Europe, Asia, Middle East, Africa
- **👥 Manager Performance** - Individual manager metrics with color-coded performance
- **🔄 Real-Time Updates** - Instant dashboard refresh when data changes
- **📋 Data Validation** - Dropdown lists for status, region, and managers
- **💾 Export Capability** - One-click CSV export of project data
- **📱 User-Friendly Interface** - Intuitive buttons and controls
- **🎨 Professional Design** - Executive-ready formatting and color coding

## 🚀 Quick Start

### Prerequisites
- Microsoft Excel 2010 or later (Windows)
- Macros enabled in Excel settings
- Basic Excel knowledge

### Installation (5 Minutes)

1. **Download** the Excel file containing the VBA code
2. **Enable Macros** when opening the file:
   - Go to File → Options → Trust Center → Trust Center Settings → Macro Settings
   - Select "Enable all macros"
3. **Open VBA Editor** (Optional) - Press `Alt+F11` to view the code
4. **Run Setup** - Press `Alt+F8`, select `SetupTracker`, and click Run
5. **Start Using** - The dashboard will auto-generate with 120+ sample projects
